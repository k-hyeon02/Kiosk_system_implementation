# Kiosk System Implementation

C언어와 `ncurses` 라이브러리를 사용하여 구현한 터미널 기반 카페 키오스크 시스템입니다.

이 리포지토리에는 C언어 소스 코드, 메뉴 데이터 파일, 그리고 프로젝트 발표 자료가 포함되어 있습니다.

---

### 🗂️ 파일 구성

* `3조.c`: 메인 C 소스 코드 파일
* `menu.csv`: 키오스크에서 사용되는 메뉴 및 가격 정보 데이터
* `C언어 최종 발표.pdf`: 프로젝트 개요 및 코드 설명이 포함된 발표 자료

### ⚙️ 주요 기능

* **터미널 UI:** `ncurses` 라이브러리를 활용한 텍스트 기반 인터페이스
* **동적 메뉴:** `menu.csv` 파일에서 메뉴 항목과 가격(S, M, L)을 동적으로 로드
* **주문 시스템:** 사용자가 메뉴를 선택하고 주문을 완료하는 기능
* **관리자 모드:** (발표 자료에 언급된) 관리자 기능 포함

### 🚀 빌드 및 실행 방법

이 프로그램은 `ncurses` 라이브러리가 필요합니다. (Linux, macOS 등 Unix 계열 환경)

1.  **의존성 설치 (Ubuntu/Debian의 경우):**
    ```bash
    sudo apt-get install libncurses-dev
    ```
    *(macOS의 경우 Xcode Command Line Tools에 기본 포함되어 있을 수 있습니다.)*

2.  **컴파일:**
    터미널에서 다음 명령어를 입력하여 컴파일합니다.
    ```bash
    gcc 3조.c -o kiosk -lncurses
    ```

3.  **실행:**
    ```bash
    ./kiosk
    ```

> **중요:** 프로그램이 `menu.csv` 파일을 올바르게 읽으려면, 컴파일된 실행 파일(`kiosk`)과 `menu.csv` 파일이 반드시 같은 디렉토리에 있어야 합니다.

### 🎮 사용 방법

* **메뉴 탐색:** 방향키 (↑, ↓)
* **항목 선택:** Enter 키
* **취소/뒤로가기:** (발표 자료에 언급된) ESC 키
